Installation Guide

1. Download the Extension

Get the latest release from our GitHub releases page.

2. Install in Chrome

Go to chrome://extensions, enable Developer Mode, and drag the downloaded ZIP file into the browser window.

3. Set Up Backend

Clone the repository and run the following commands:

yarn setup
yarn start

4. Start Mocking

Open Chrome DevTools, navigate to the "Mockey Patcher" panel, and start capturing requests.
